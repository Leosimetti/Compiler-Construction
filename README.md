# Compiler-Construction  
To track the progress of our CC course project  

## How to run my lexer
**Step 1:** cd <path/to/github>  
**Step 2:** $ git checkout Ozzie  
**Step 3:** $ cd lexer  
**Step 4:** ensure the test suite you want to lex is in 'src/suite.txt'  
**Step 5:** $ make  
**Step 6:** $ ./main.out
